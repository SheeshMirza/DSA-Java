## Binary Search

### Input: 
- Target (integer target) and Sorted List (integer[] array)

### Output:

- Return (IF FOUND): ```Index``` of the array (or list), whose value is equal to the target
- Return (IF NOT FOUND): ```-1```

### Time Complexity:
- O ( log n )

### Example:
```yaml
Search.BinarySearch(int target, int[] array);
```
